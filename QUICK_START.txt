--------------------------------------------------------------------------------
OPTION 2: AUTO-SEARCH FOR VALID STATIONS
--------------------------------------------------------------------------------

python3 stofs2d_obs_prototype.py

OUTPUT: stofs2d_comparison_prototype.png

This tries multiple stations automatically until it finds one with data.

--------------------------------------------------------------------------------
CUSTOMIZE STATION
--------------------------------------------------------------------------------

Edit stofs2d_demo_quick.py:

  station_info = fort61.get_station_info(30)  <-- Change 30 to any index
  model_data = fort61.get_station_data(30)    <-- Change 30 to match

Save and run again:
  python3 stofs2d_demo_quick.py

--------------------------------------------------------------------------------
CHANGE DATUM (Vertical Reference)
--------------------------------------------------------------------------------

Edit stofs2d_demo_quick.py:

  datum = 'MSL'     <-- Change to 'MLLW', 'NAVD88', or 'STND'

Save and run again.

--------------------------------------------------------------------------------
LIST AVAILABLE STATIONS
--------------------------------------------------------------------------------

python3 -c "from stofs2d_obs_prototype import Fort61Reader; \
f = Fort61Reader('fort.61.nc'); \
f.list_stations(n=20); \
f.close()"

This shows first 20 stations with their coordinates.

--------------------------------------------------------------------------------
COMMON ISSUES & FIXES
--------------------------------------------------------------------------------

Issue: "No module named 'netCDF4'"
Fix:   pip3 install --break-system-packages netCDF4 matplotlib numpy searvey pandas geopandas

Issue: "No CO-OPS stations found"
Fix:   Try a different station index or increase search radius

Issue: "No observation data"
Fix:   Station may not support that datum or time range. Try different datum.
