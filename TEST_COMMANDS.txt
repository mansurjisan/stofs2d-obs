=================================================================
STOFS2D-OBS: Test Commands Quick Reference
=================================================================

## BASIC COMMANDS

Run all unit tests (fast):
  pytest tests/ -v -m "unit and not slow"

Run all tests (including slow integration tests):
  pytest tests/

Run specific test file:
  pytest tests/test_fort61.py -v
  pytest tests/test_comparison.py -v
  pytest tests/test_cli.py -v

## COVERAGE COMMANDS

Run with coverage report:
  pytest tests/ --cov=stofs2d_obs --cov-report=term-missing

Generate HTML coverage report:
  pytest tests/ --cov=stofs2d_obs --cov-report=html

View HTML coverage (in browser):
  explorer.exe htmlcov/index.html

## FILTERING TESTS

Run only unit tests:
  pytest -m unit

Run only integration tests:
  pytest -m integration

Run only slow tests:
  pytest -m slow

Run specific test by name:
  pytest tests/test_fort61.py::TestFort61Reader::test_init_opens_file

## DEBUGGING

Run with verbose output:
  pytest tests/ -vv

Run with detailed traceback:
  pytest tests/ --tb=long

Run and stop at first failure:
  pytest tests/ -x

Run and enter debugger on failure:
  pytest tests/ --pdb

## OUTPUT CONTROL

Show print statements:
  pytest tests/ -s

Quiet mode (less output):
  pytest tests/ -q

Show test durations:
  pytest tests/ --durations=10
